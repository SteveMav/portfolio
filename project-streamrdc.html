StreamRDC est une plateforme web de gestion et partage d'abonnements Netflix en République Démocratique du Congo. Connectez-vous, payez via M-Pesa, Orange Money ou Airtel Money, et obtenez votre abonnement en quelques secondes.

DJANGO JAVASCRIPT TAILWIND CSS MYSQL

À propos du projet

StreamRDC est une plateforme web de gestion et partage d'abonnements Netflix en République Démocratique du Congo. Elle permet aux utilisateurs de souscrire à un abonnement Netflix en toute simplicité via les moyens de paiement locaux (M-Pesa, Orange Money, Airtel Money).

Le système offre une livraison instantanée des accès après validation du paiement, un programme de parrainage et récompenses, et une qualité vidéo maximale en 4K Ultra HD. L'accent a été mis sur la facilité d'utilisation et la rapidité du processus.

Fonctionnalités clés

  • Inscription et connexion rapide (email ou Google)
  • Paiement via M-Pesa, Orange Money et Airtel Money
  • Livraison instantanée de l'accès Netflix après validation
  • Programme de parrainage et récompenses
  • Qualité Vidéo Max 4K Ultra HD
  • Dashboard d'administration pour la gestion des utilisateurs et transactions
  • Interface responsive avec Tailwind CSS

Stack Technique

Backend : Django (Python) – Framework web puissant avec ORM intégré, gestion des utilisateurs et API REST.
Frontend : JavaScript + Tailwind CSS – Interface moderne et reactive avec des composants stylisés.
Base de données : MySQL – Base de données relationnelle pour la gestion des données utilisateurs et des transactions.
Déploiement : Serveur de production avec configuration optimisée.
